# Web frontend

Next.js app for the municipal election party-program comparison site.

The app reads `src/data/parties.json`, which is generated by the pipeline:

```bash
# from the project root
python3 pipeline/transform.py
```

Then build or run locally:

```bash
npm run dev    # development
npm run build  # production build
```

For deployment see the root `DEPLOYMENT.md`.
For reproducing this for a different municipality see `REPRODUCE.md`.
